Darpan is a reconciliation workspace. It connects source systems, describes source data with schemas, runs comparisons, applies RuleSets, and preserves the output needed to explain each difference.

The docs are organized around the path of a run: setup records, comparison, generated output, and follow-up changes to source data, schema, or rules.
